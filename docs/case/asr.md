# AI语音识别编程



| 项目名称 | 学习目标 |
|---------|----------|
| M5 ASR语音操控小车 | 1. 掌握M5 ASR语音识别模块与MicroBlocks的串口通信<br>2. 学习语音命令词训练与识别（如前、后、左、右、停止）<br>3. 实现语音控制小车运动（结合TT电机或编码电机）<br>4. 学习多条件分支结构与语音指令解析编程 |
| DFRobot ASR语音控制开关门 | 1. 掌握DFRobot ASR模块的唤醒词与自定义命令设置<br>2. 理解语音识别结果与舵机控制信号的映射关系<br>3. 实现语音控制舵机旋转（模拟开关门动作，0°~90°或0°~180°）<br>4. 学习状态变量保存门当前开闭状态，防止重复触发 |
