# MicroBlocks案例展示



| 案例 | 案例目标 |
|------|----------|
| [LED和屏幕的编程](led.md) | 1. 掌握控制单个LED和流水灯效果<br>2. 学习使用点阵或OLED屏幕显示图形、文字和数字<br>3. 制作倒计时器、电子骰子或简单动画<br>4. 理解数字输出与模拟输出（PWM调光）的区别 |
| [数字和几何编程](math.md) | 1. 使用海龟绘图绘制三角形、四边形和多边形<br>2. 学习角度、边长、坐标等几何概念<br>3. 结合变量和循环生成螺旋、星形等复杂图形<br>4. 掌握数学运算积木（加减乘除、取余、随机数、映射） |
| [代码演奏音乐编程](music.md) | 1. 使用蜂鸣器播放单音和简单旋律<br>2. 学习MIDI协议与音调库，切换不同乐器音色<br>3. 编程演奏音阶、和弦（大三/小三）<br>4. 理解音符时长、节拍与等待积木的配合 |
| [Input传感器编程](input.md) | 1. 掌握按钮、旋转电位器、超声波、温湿度等传感器数据读取<br>2. 理解数字信号（按钮）与模拟信号（电位器）的区别<br>3. 实现传感器控制输出设备（如超声波控制LED亮度）<br>4. 学习多条件分支结构与变量存储传感器数值 |
| [舵机和电机编程](servo.md) | 1. 掌握180°舵机角度控制与360°舵机连续旋转<br>2. 学习TT直流电机的PWM调速和正反转控制<br>3. 了解霍尔编码电机测速与闭环控制原理<br>4. 制作遥控小车或机械臂等运动系统 |
| [语音识别模块编程](asr.md) | 1. 掌握M5 ASR或DFRobot ASR模块与MicroBlocks的通信<br>2. 学习训练自定义语音命令词（如前、后、左、右、开门）<br>3. 实现语音控制LED、舵机、电机等执行器<br>4. 学习串口数据解析与多条件命令分支响应 |
|  [AI摄像头识别编程](aicamera.md) | 1. 掌握Sentry或K210 AI摄像头与MicroBlocks的通信协议<br>2. 学习识别交通标志、数字卡片、颜色或人体特征<br>3. 实现视觉反馈控制（如红灯停、识别数字后播放音调）<br>4. 理解识别结果（标签、置信度）的解析与逻辑判断 |
| [通信模块编程](comm.md) | 1. 掌握蓝牙广播实现双板无线遥控<br>2. 学习Wi-Fi连接互联网获取NTP时间<br>3. 理解MQTT协议实现物联网数据上传与远程控制<br>4. 学习红外遥控解码与开关控制逻辑 |
