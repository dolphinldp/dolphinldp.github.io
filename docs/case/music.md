# 图形积木演奏音乐
 

| 项目名称 | 学习目标 |
|---------|----------|
| 蜂鸣器和音调库 | 1. 掌握无源蜂鸣器的发声原理与控制方法<br>2. 使用音调库生成不同频率的声音<br>3. 编写简单旋律（如《小星星》）<br>4. 学习“播放音调”积木与时长控制 |
| MIDI音乐和MIDI库 | 1. 理解MIDI协议与乐器编号（Program Change）<br>2. 使用MIDI库选择不同乐器音色<br>3. 编写多轨道MIDI序列<br>4. 学习MIDI音符编号与播放事件积木 |
| 音符音程与和弦 | 1. 理解音程（半音/全音）与和弦（大三/小三）理论<br>2. 使用音符积木构建C大调音阶<br>3. 编程播放三和弦（C、F、G）<br>4. 学习自定义积木封装和弦函数 |
