# 通信模块编程




| 项目名称 | 学习目标 |
|---------|----------|
| 蓝牙广播遥控小车 | 1. 掌握MicroBlocks蓝牙广播（BLE）的发送与接收机制<br>2. 学习使用两个MicroBlocks板卡进行无线通信（遥控器→小车）<br>3. 实现方向按键广播控制小车前进、后退、转弯、停止<br>4. 学习数据打包、广播地址匹配与防冲突设计 |
| Wi-Fi通信获取时间 | 1. 掌握MicroBlocks Wi-Fi扩展板的网络连接配置（SSID/密码）<br>2. 学习通过NTP协议从互联网获取当前日期与时间<br>3. 实现在点阵或LCD显示屏上显示实时时钟（时:分:秒）<br>4. 理解时区转换与网络请求积木的使用方法 |
| MQTT通信 | 1. 理解MQTT协议（发布/订阅模型）与物联网基本概念<br>2. 学习配置MQTT Broker服务器地址与端口<br>3. 实现传感器数据上传（如温湿度）或接收远程控制指令<br>4. 学习主题（Topic）设计与消息负载解析 |
| 红外遥控开关门 | 1. 掌握红外接收模块解码红外遥控器按键码值<br>2. 学习使用红外键值触发舵机转动（模拟开门/关门动作）<br>3. 实现单键切换状态（按一次开门，再按一次关门）<br>4. 学习红外信号的学习与存储功能（适配不同遥控器） |
