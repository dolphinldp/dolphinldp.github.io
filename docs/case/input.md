# INPUT传感器编程


| 项目名称 | 学习目标 |
|---------|----------|
| 按钮控制 | 1. 掌握按钮的数字信号读取方法<br>2. 理解上拉电阻与电平变化<br>3. 实现按钮控制LED或蜂鸣器<br>4. 学习“当按下时”事件触发编程 |
| 旋转电位器 | 1. 理解模拟输入与ADC转换<br>2. 读取电位器的模拟值（0-4095或0-1023）<br>3. 控制LED亮度或舵机角度<br>4. 学习映射（map）积木的使用 |
| 超声波测距 | 1. 掌握超声波模块的Trig/Echo引脚控制<br>2. 理解声波测距原理与时间差计算<br>3. 实现距离检测与报警功能<br>4. 学习“重复执行”与条件判断结合应用 |
| 温湿度传感 | 1. 掌握DHT11/DHT22单总线通信协议<br>2. 读取温度与湿度数据<br>3. 在点阵或串口显示环境数据<br>4. 学习数据解析与延时控制 |
| 巡线模块 | 1. 理解红外反射原理与黑白检测<br>2. 读取双路/多路巡线传感器信号<br>3. 实现小车沿黑线行驶逻辑<br>4. 学习“如果-否则”多条件分支结构 |
| 红外接收模块 | 1. 掌握红外解码原理与NEC协议<br>2. 读取遥控器按键键值<br>3. 实现遥控控制LED、电机等执行器<br>4. 学习变量存储键值与事件响应编程 |
